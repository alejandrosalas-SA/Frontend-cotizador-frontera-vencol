# Instalación de Dependencias del Sistema de Autenticación

## Ejecutar estos comandos en orden:

```bash
# 1. Dependencias de producción
npm install react-hook-form @hookform/resolvers zod @tanstack/react-query jwt-decode

# 2. Componentes UI de shadcn (opcional, si no están instalados)
npx shadcn@latest add input label form card alert
```

## Verificar instalación

Después de instalar, verificar que las dependencias estén en `package.json`:
- react-hook-form
- @hookform/resolvers
- zod
- @tanstack/react-query
- jwt-decode
